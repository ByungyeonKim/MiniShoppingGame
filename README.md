# MiniShoppingGame

- 아이템별로 나누어 볼 수 있는 미니쇼핑게임입니다.
- 데이터(아이템)를 json 파일에서 불러와(fetch)사용했습니다.

##### 기술 스택

- HTML
- CSS
- JavaScript
- JSON

##### 사용한 에디터

- VScode
